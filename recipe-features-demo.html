üéØ How to Test the Features

1. Recipe Import/Export

  1. Navigate to Inventory ‚Üí Recipes tab
  2. Click the "Import/Export" dropdown button
  3. Download CSV template to see the format
  4. Try importing recipes from CSV or JSON files
  5. Export existing recipes to CSV or JSON

2. POS Integration

  1. Create a recipe with ingredients from inventory
  2. Link the recipe to a product in POS
  3. Make a sale in POS with that product
  4. Check inventory - ingredients should be automatically deducted
  5. View usage logs in recipe analytics

3. Smart Alerts

  1. Set minimum thresholds for inventory items used in recipes
  2. Reduce stock below threshold levels
  3. View the Recipe Ingredient Alerts component
  4. See which recipes are affected by low stock
